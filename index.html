Geldstromen instellen

Je data wordt opgeslagen in je eigen GitHub repository — gratis en privé. Eenmalig instellen, daarna werkt alles automatisch.

1
Ga naar github.com/settings/tokens/new, geef het token een naam (bijv. "Geldstromen"), selecteer alleen repo scope, en klik Generate token.
2
Kopieer het token (begint met ghp_) en plak het hieronder.
3
Vul je GitHub gebruikersnaam en de naam van je repository in (bijv. Financieel-dashboard).
