Geldstromen instellen

Je data wordt opgeslagen in je eigen GitHub repository — gratis en privé. Eenmalig instellen, daarna werkt alles automatisch.

1
Ga naar github.com/settings/tokens/new, geef het een naam, vink alleen repo aan, klik Generate token.
2
Kopieer het token (begint met ghp_) en plak hieronder.
3
Vul je GitHub gebruikersnaam en de exacte naam van je repository in.
